# 🚀 Autonomous Software Architecture Planning & Evolution using Agentic AI

## 📌 Overview
This project presents an **AI-powered decision-support system** that automates **software architecture design, analysis, and evolution** using an **Agentic AI multi-agent framework**.

It helps developers:
- Design architecture for new systems (Greenfield)
- Analyze and improve existing systems (Brownfield)
- Detect architectural risks early
- Get intelligent recommendations before implementation

---

## 🎯 Problem Statement
Traditional software architecture evaluation tools operate **after development**, leading to:
- Late detection of issues  
- Increased refactoring cost  
- Poor scalability and maintainability  

This project solves this by providing a **proactive, AI-driven architecture planning system**.

---

## 💡 Key Features

### 🔹 Greenfield Mode (New Systems)
- Accepts functional & non-functional requirements
- Suggests:
  - Architecture patterns (Microservices, MVC, Layered)
  - Modular decomposition
- Generates optimal architecture blueprint

---

### 🔹 Brownfield Mode (Existing Systems)
- Upload existing codebase
- Performs:
  - AST parsing
  - Dependency analysis
- Detects:
  - Tight coupling
  - Low cohesion
  - Anti-patterns
- Suggests refactoring improvements

---

### 🔹 AI Capabilities
- Multi-agent decision-making system
- Architecture analysis & planning agents
- Quality evaluation & suggestion engine
- Explainable recommendations

---

### 🔹 Visualization & Reports
- Dependency graph visualization
- Risk analysis report
- Architecture recommendations

---

## 🏗️ System Architecture
User Input → Frontend → Backend (FastAPI)
→ Agent Orchestrator
→ Multi-Agent System
→ Analysis + Suggestions
→ Visualization & Output


### Layers:
- **Frontend:** React.js + Tailwind CSS + D3.js  
- **Backend:** FastAPI (Python)  
- **AI Layer:** LangChain + LangGraph  
- **Analysis Tools:** AST Parsing, Dependency Graph
